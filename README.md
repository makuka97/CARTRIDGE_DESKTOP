CARTRIDGE v0.4.1 brings native emulation for Nintendo 64, GameCube, and Wii. N64 runs through Mupen64Plus with full-speed dynarec, save states, and controller support, close the emulator and you're back in the library automatically. GameCube and Wii run through Dolphin with its full GUI, save states, and controller config. This release also expands the supported system library to include Virtual Boy, Sega Master System, Game Gear, Sega CD, TurboGrafx-16, Neo Geo Pocket, WonderSwan, ColecoVision, and Atari Lynx.

Everything from v0.4.0 is still here, box art scraping via SteamGridDB, save states for all EmulatorJS cores, drag-and-drop ROM import, keyboard remapping, and controller hot-plug support. As always, CARTRIDGE is 100% offline after install with all cores bundled.
